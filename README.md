# Inteligencia Artificial

## Introducción

La Inteligencia Artificial (IA) es un área multidisciplinaria que, a través de ciencias como la computación, la lógica y la filosofía, estudia la creación y diseño de entidades capaces de resolver problemas por sí mismas, tomando como paradigma la inteligencia humana.
